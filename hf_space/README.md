# Property Classifier
AI-Powered Building Environment Analysis using ResNet101V2 Transfer Learning.

## Classes
- Exterior Facade — Building exteriors and facades
- Office Interior — Office rooms and workspaces  
- Warehouse — Storage and industrial facilities
- HVAC Pipeline — Mechanical systems and pipelines

## Model
- Backbone: ResNet101V2 (ImageNet pretrained)
- Trainable parameters: 24,640
- Input size: 224 x 224
